# diegomrsantos.github.io

Minimal static site for GitHub Pages.

## Files

- `index.html`: single-page About section
- `styles.css`: layout and visual styling

## Publish

1. Create a GitHub repository named `diegomrsantos.github.io`.
2. Push these files to the `main` branch of that repository.
3. In the repository settings, enable GitHub Pages from the root of `main` if GitHub does not publish it automatically.

## Local preview

Open `index.html` directly in a browser, or serve the folder locally:

```sh
cd /Users/diegomrsantos/diegomrsantos.github.io
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
